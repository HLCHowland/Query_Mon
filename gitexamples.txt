Got to clone and clone the repository wherever you want it.


$git pull -gets any new files or changes and updates your directory

The two below must be done together
$git commit -m "your message here"
$git push -pushes commited changes to the repo

$git status -shows current files that are not in the repo



Adding Files to Repo
	1. Go to git directory you cloned or have been working in via the terminal.
	2. $git add <file name here>
	3. $git commit -m "your message here"
	4. $git push origin master
